# AhhbitTracker Frontend

React frontend for AhhbitTracker habit tracking dApp.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```
or

```bash
cd frontend
node node_modules/vite/bin/vite.js
```
Opens at http://localhost:3000

## Build

```bash
npm run build
```

Output in `dist/`

## Features

- Wallet connection (Leather, Xverse, Asigna)
- Habit creation
- Habit list view
- Daily check-ins
- Streak tracking
- Stake withdrawal
- Bonus claims

## Tech Stack

- React 18
- TypeScript
- Vite
- @stacks/connect
- @stacks/transactions
