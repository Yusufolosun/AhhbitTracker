# Analytics Suite

Utilities for monitoring contract usage and metrics.

## Available Tools

### Streak Analyzer
```bash
npm run analytics:streaks
```

### Pool Tracker
```bash
npm run analytics:pool
```

### User Activity
```bash
npm run analytics:activity
```

### Full Dashboard
```bash
npm run analytics:dashboard
```

## Data Files

- `pool-history.json` - Historical pool balance snapshots
- Generated automatically when running pool tracker

## Usage

Run regularly to monitor:
- Contract growth
- User engagement
- Pool accumulation
- Streak patterns
